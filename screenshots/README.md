# Screenshots Directory

This directory contains screenshots and images for the README.md file.

## How to Add Screenshots

1. **Take screenshots** of your application:
   - Main interface
   - YouTube Downloader tab
   - Audio Clipper tab
   - Audio Merger tab
   - Any other features you want to showcase

2. **Save screenshots** in this directory with descriptive names:
   - `main-interface.png`
   - `youtube-downloader.png`
   - `audio-clipper.png`
   - `audio-merger.png`

3. **Add to README.md** using markdown image syntax:
   ```markdown
   ![Alt text](screenshots/filename.png)
   ```

4. **For better display**, you can also use HTML in markdown:
   ```markdown
   <img src="screenshots/filename.png" alt="Description" width="800">
   ```

## Image Formats Supported

- PNG (recommended for screenshots)
- JPG/JPEG
- GIF (for animated demonstrations)
- SVG (for logos/icons)

## Best Practices

- Keep file sizes reasonable (< 1MB per image)
- Use descriptive filenames
- Consider using a tool like [TinyPNG](https://tinypng.com/) to compress images
- Maintain consistent image widths in your README (e.g., 800px)
